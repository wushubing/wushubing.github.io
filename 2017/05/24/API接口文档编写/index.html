API接口文档编写

Veröffentlicht am 2017-05-24 |

用户登录

验证用户名密码

接口：

POST /login

请求参数：

名称 类型 定义 必需 默认值 说明
username string 用户名 “”
password number 密码 “”
cb number 是否记住密码 0 不记住

响应：

成功：200

响应格式：JSON

{
     $.ajax({
            type: "POST",
            url: "", //登录事件处理
            data: {
                "username": {string},
                "password": {number},
                "cb": {number}
            }
}

缓存配置：使用ETag进行缓存。

参数不合法：409

参考标准409响应

banner图片

用于获取图片相关信息

接口

GET /bannerimage

请求参数

名称 类型 定义 必需 默认值 说明
src string 查询关键词 “#” 图片链接
alt string 图片说明 “” 图片说明
target string 跳转链接 “#” 图片点击跳转
id string 名称 “” “id”

响应：

成功：200

响应格式：JSON

{
    data: [
        {
            "src": {string},
            "alt": {string},
            "target": {string},
            "id": {string}
        },
        ...
    ]
}

参数不合法：404

参考标准404响应
